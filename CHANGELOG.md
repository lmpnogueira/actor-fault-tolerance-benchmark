## [1.0.0] - 2026-XX-XX

Initial public release accompanying the paper:

> **Benchmarking Fault-Tolerance Characteristics of Actor-Based Runtimes**

### Added

- Language-independent benchmarking framework for evaluating actor-based runtimes under recurring transient failures.
- Three functionally equivalent implementations:
  - Elixir / BEAM
  - Scala / Akka
  - Go / Proto.Actor
- Automated benchmark configuration generator.
- Benchmark automation scripts.
- External statistical processing pipeline.
- Experimental datasets and plotting utilities.
- Documentation and reproducibility guide.

### Notes

This release corresponds to the implementation and experimental artefacts used to produce the results reported in the accompanying publication.
